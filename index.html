Welcome to my page for documentation and misc tools.

So this is the site I plan on using and distributing to allow anyone to easily build their character stat arrays and a location for quick and easy dice rolling. It is a work in progress and will continue to imrpove over time, but as it currently stands, the site works for the basic goals. Eventually I plan to add some additional features like adding in racial bonuses and tracking the rolls of individual dice instead of just outputting the full amount. I might also add a few documents for character building and guides on optimizing characters.
